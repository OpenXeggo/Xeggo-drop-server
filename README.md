# Linkdrop Relayer Server

An external server, which calls factory smart contract on behalf of the receiver. The relayer server pays for gas instead of receiver.

## Installation

### Install dependencies

```bash
yarn install
```

### Run server

```bash
yarn start
```

### Deploy server

```bash
yarn deploy
```
